--
Apa pengertian dari character set?

Jawab:
Character set (biasa disingkat sebagai charset) adalah kumpulan karakter yang bisa dipahami oleh komputer.

--
Apa istilah lain dari database?

Jawab:
SCHEMA

--
Tuliskan query untuk menampilkan character set di MYSQL!

Jawab:
Untuk melihat apa saja character set yang tersedia, bisa menjalankan query SHOW CHARACTER SET;

--
Tuliskan query untuk melihat collation yang terdapat di MYSQL!

Jawab:
Untuk melihat apa saja collation yang didukung oleh MySQL bisa menjalankan query SHOW COLLATION;

--
Sebutkan 5 tipe data numeric pada MYSQL!

Jawab:
1. TINYINT
2. SMALLINT
3. MEDIUMINT
4. INT
5. BIGINT 

--
Sebutkan 5 tipe data string pada MYSQL!

Jawab:

4. TEXT
5. MEDIUMTEXT

--
Sebutkan 5 tipe data date pada MYSQL!

Jawab:
1. DATE
